# The Microservices architecture 
The Microservice architecture is based on the following main objectives.. 

* the service must be resilient, 
* self-healing 
* and can evolve


# How to make a micro service resilient:
* Idempotent
* Timeout
* Deadline
* Retries
* Circuit breakers
* Rate limiters

# Handling requests
* Choreography
* Orchestration
